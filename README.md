# 极客护眼提醒 (Eye Reminder)

这是一个基于原生 Win32 C 语言编写的极致轻量、极客纯粹的桌面护眼与专注定时工具。

## 🌟 核心理念与特性
为了追求绝对的高效、低资源占用与无打扰沉浸感，本程序具有以下特色：

1. **极致轻量体积**：纯 Win32 API 编写，不依赖任何第三方臃肿库（如 Qt/Electron/WPF/Webview），编译后可执行文件不到 1MB，内存占用极少。
2. **底层纯净架构 (0 注册表污染)**：
   - 彻底脱离对 Windows 注册表的硬编码依赖。
   - 配置项（工作时长等）实时序列化保存到本目录下的 `config.ini`，实现绝对的绿色便携（Portable）。
   - （仅“开机自启动”功能使用系统规范的 Run 注册表键值记录）。
3. **原生悬浮胶囊与多显示器自适应**：
   - 桌面右下角的倒计时胶囊采用了原生 Windows DWM 半透明及边角圆润特性，彻底抛弃了易发糊的绘制策略，完美拥抱 `LOGPIXELSY` 动态获取的高分屏 DPI 缩放。
   - 真正的**多屏流式追踪**：胶囊窗口不会死板地停留于主屏幕，而是基于底层的 `MonitorFromPoint` 算法，实时、智能地附属在你鼠标当前所在的活跃显示器右下角（且已避让常见输入法工具栏高度）。
4. **硬核单例防护与会话拦截**：
   - 应用引入底层的 `CreateMutexW` 系统互斥锁，全局严格保障**有且仅有一个实例**存活，拒绝重影倒计时。
   - 拦截系统 `WM_WTSSESSION_LOCK`，当你按下 `Win + L` 锁屏离开电脑时，护眼倒计时将被“物理挂起”并暂停，解锁后无缝恢复。
5. **绝对纯粹的专注模式**：
   - 支持从托盘或直接双击胶囊唤醒设置/右键菜单。
   - 包含一键“专注模式（2小时）”，按下后不仅屏幕再无强制锁屏打扰，托盘提示与桌面数字胶囊也会被瞬间“冷冻”在按下那一刻的时间（不会跳动），还你绝对清净。并在2小时后自动退出专注并提示。

## 🚀 编译说明
本项目使用了原生 MinGW-w64 (GCC) 编译器直接构建。为了将资源（如自定义图标）与 C 语言核心混合打包，请直接运行同级目录的构建脚本。

双击执行 `build.bat` 即可一键生成最新的 `eye_reminder.exe`。
> （编译参数中已经连接了诸如 `-lgdi32`, `-lcomctl32`, `-ldwmapi`, `-lwtsapi32` 等核心系统组件。）

## ⚙️ 使用说明
- **运行方式**：双击 `eye_reminder.exe` 执行。
- **配置与设置**：双击系统右下角托盘图标，或**双击桌面右下角的悬浮胶囊**即可打开参数配置界面（支持调整工作轮转时长与开机自启）。
- **右键菜单**：右键点击托盘图标 或 **右键点击悬浮胶囊**，即可呼出主菜单（可推迟、开启/结束专注模式、退出）。
- **绿色卸载**：如果不需要开机开机启动了，在设置里面取消勾选保存，然后直接 Shift+Delete 删除整个文件夹即可！干干净净，绝无驻留。
